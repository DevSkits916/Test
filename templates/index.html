About

Sora Invite Code Hunter monitors public surfaces for mentions of potential Sora 2 invite codes. It respects rate limits, avoids private data, and is intended for ethical monitoring only.

  • Sources: curated Reddit queries, RSS feeds, generic HTML pages, and Twitter/X searches.
  • Heuristics: strict length bounds, denylist filtering, digit requirement, and repeated-pattern checks.
  • Usage: do not resell harvested information and respect the terms of each source.
